# SWARMSY Anti-Blandness Gate

Use this gate before approving any idea, post, campaign, or stunt.

## Gate Questions

1. Would this stop someone scrolling?
2. Would someone repeat this phrase?
3. Could someone photograph or share this?
4. Does this create curiosity?
5. Does this fit the lore?
6. Does it create a next action?
7. Is it honest?
8. Is it legal-safe?
9. Is it different from a normal brand post?
10. Would this embarrass the identity by being too soft?

## Scoring

- **1 = dead content**
- **2 = generic**
- **3 = usable but weak**
- **4 = strong**
- **5 = living icon signal**

Score each question from 1-5. Use the average score as final grade.

## Pass / Fail Rule

- **Below 4.0**: fail and revise.
- **4.0-4.4**: usable, revise once for sharper signal.
- **4.5-5.0**: approved for release.

If an idea scores below 4, SPARKY must revise it.

## “Make it less Disney” Revision Process

1. remove generic positivity
2. add tension
3. add symbol
4. add clue
5. add stronger visual
6. add sharper phrase
7. add audience action
8. add proof hook
9. remove fake hype
10. keep legal-safe

## Automatic Rejection Triggers

Reject immediately if any concept includes:

- fake claims/proof/hype/outrage/community
- spam behavior or manipulation tactics
- illegal, reckless, harassment, vandalism, or evasion guidance

## Output Format

For each concept provide:

- score table (10 questions)
- final score
- revision notes (if below 4)
- approved version (legal-safe)
