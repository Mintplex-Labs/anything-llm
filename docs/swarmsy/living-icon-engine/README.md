# SWARMSY Living Icon Engine

This folder contains the SWARMSY Living Icon Engine prompt tree.

DIZ-A-REMIX is intended to become a project-manager-first AnythingLLM fork. These files define the operating doctrine for an AI project manager that helps users build identity, lore, campaigns, PR assets, product ladders, content systems, legal-safe activation plans, and daily momentum.

## Core Rule

No fake claims.  
No fake bots.  
No fake engagement.  
No invented proof.  
No generic brand-app fluff.  
No reckless illegal instructions.

Shock the attention system, not the law.

## Intended Product Direction

AnythingLLM is only the base. SWARMSY should become the project manager layer.

The final app should help a user move through:

1. Intake
2. Identity
3. Lore
4. Visual world
5. Product ladder
6. Content and PR machine
7. Legal-safe activation
8. 30-day campaign
9. Day 1 campaign pack
10. Final AI project manager memory lock

After step 10, the AI should be able to continue with Day 2, Week 2, next drop, next campaign, next review, or next PR push without needing the original prompt files again.
