# STREET_ARTIST_IDENTITY_PACK_V1.md

## 1. Identity Snapshot

Create a concise snapshot:

- Artist name
- Identity mode
- Public/private boundary
- One-line pitch
- Core message
- Core audience
- Main visual mood
- Main commercial direction
- Main project risk

If the artist name is not obvious yet, generate options in Section 3 and mark this as “Pending Final Name”.

---

## 2. User Truth Summary

Summarise the real user without exposing private details.

Include:

- background energy
- lived experience themes
- personality
- humour
- anger
- values
- creative drive
- what makes this user different
- what must not be exaggerated

Be direct.

Do not flatter without evidence.

---

## 3. Artist Name System

Generate 10 artist name options.

For each name include:

- meaning
- vibe
- strengths
- weaknesses
- visual potential
- whether it suits Face Identity Mode, Hidden Identity Mode, or both

Then choose:

- Top 3 names
- Final recommended name
- Backup name if the final name is unavailable

Do not claim the name is legally available.

Tell the user the name must be checked for domain, social handle, company, and trademark availability before commercial use.

---

## 4. Identity Mode Build

If the user chose Face Identity Mode, create:

- public-facing identity angle
- how much real life to show
- how to use face/camera presence
- how to build trust
- what to avoid oversharing
- how to turn real story into collector value
- how to stay safe while being visible

If the user chose Hidden Identity Mode, create:

- alias/persona
- mask or anonymity concept
- public myth
- trust-building system without showing identity
- privacy rules
- reveal/no-reveal strategy
- how to keep the mystery consistent
- how to stop the identity becoming gimmicky

If the user was undecided, recommend one mode and explain why.

---

## 5. Origin Story

Create two origin stories:

### Short Origin Story

For social bios, short posts, captions, marketplace profiles, or press snippets.

### Long Origin Story

For website, collector pages, interviews, artist statements, and launch material.

Separate:

- true-life elements
- symbolic/lore elements
- marketing language

Do not present invented mythology as literal fact.

---

## 6. Core Lore Bible

Create the first lore bible.

Include:

- core myth
- central conflict
- what the artist fights against
- what the artist protects
- recurring symbols
- recurring phrases
- recurring colours
- recurring objects
- character/mascot/mask ideas if relevant
- hidden meanings
- world rules
- what must stay consistent
- what must never be used
- how future artwork can connect into the lore

This must become reusable canon for future AI agents.

---

## 7. Manifesto

Write a powerful artist manifesto.

It must explain:

- why the artist exists
- what the work stands for
- what the work rejects
- why identity matters
- why people should care
- why collectors are buying into more than decoration
- why now matters

Tone must match the user’s voice and privacy mode.

Do not make it generic.

---

## 8. Brand Voice Guide

Create a voice guide.

Include:

- tone
- attitude

- humour style
- emotional level
- anger level
- collector-facing voice
- fan-facing voice
- press-facing voice
- words to use
- words to avoid
- phrases the artist might say
- phrases the artist should never say
- 5 short caption examples
- 3 long caption examples
- 3 collector-facing post examples

---

## 9. Visual Identity Guide

Create a visual guide.

Include:

- colour palette direction
- texture direction
- logo concept

- symbol system
- typography mood
- poster style

- canvas/original artwork style
- print style

- photography style
- social grid style
- packaging style
- exhibition/pop-up style

Use the handoff to create an original direction.
