- thank supporters
- show behind-the-scenes
- move toward product/drop

If Day 1 is quiet:

- improve hook
- simplify CTA
- create process content
- ask direct questions
- test new caption angle
- avoid panic rebrand

If Day 1 gets criticism:

- assess if useful
- do not overreact
- correct factual issues
- keep identity stable
- respond only if strategic
