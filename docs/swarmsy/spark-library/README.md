# SWARMSY Spark Library

SPARKY needs a “spark” library of proven cultural patterns so it can build systems, not just posts.

**SPARKY does not copy icons. SPARKY studies how icons become systems.**

- The goal is to learn reusable patterns, not copy any brand, artist, game, studio, or protected IP.
