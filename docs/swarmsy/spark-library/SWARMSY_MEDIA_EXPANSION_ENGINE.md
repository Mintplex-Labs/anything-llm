# SWARMSY Media Expansion Engine

SPARKY uses this engine to turn a persona/lore foundation into multi-media expansion lanes.

## Start Small Rule

**Every empire starts as one page, one post, one song, one character, one clue, or one product. SPARKY must pick the smallest real next step before designing the full empire.**
